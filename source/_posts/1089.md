7月份投资记录总结。

本月市场氛围高涨，上半月爆涨，下半个月平盘震荡，还带了两次单日暴跌，两次都进行了小额加仓，持仓组合运行稳定，除了收入结余转定投，本月对白酒进行了止盈，另外一方面工作繁忙，操作频率更加减少。

# 持仓情况

当前仓位稳定在半仓左右。以现在的大盘点位来看很难再大幅加仓，仓位参考基本取2750-3750区间，中间的3250对应半仓。

详细持仓情况：

**场内暂定(2020.5)：**
1. 轮动运作，底仓2~4层
2. 跌1%/2%/3%买入，单次0.5/1/2层
3. 涨1%/2%/3%卖出，单次0.5/1/2层
4. 早盘尾盘操作，盘中不操作

**场外暂定(2020.6)：**
1. 半仓(50-75%)运作
2. 每跌5%(单日2%/3%)加仓0.5/1/2/3/4层(最多10层)
3. 每涨7%(单日3%/5%)减仓0.5/1/2/3/4层
4. 主动基金长持(1-2年)，行业指数基金涨跌幅轮动
5. 连续无加仓点时单日跌超1%触发结余定投

**最大回撤(2020.5)：**
1. 场内底仓较低，操作灵活，可以接受较大回撤
2. 场外底仓5成，计划按照跌幅加仓4次，最大可接受20%回撤，此时浮亏11.71%

**计划仓位(2020.5)：**
1. 场内股票 上限5%
2. 场内基金 目标15%/上限30%
3. 场外基金 目标30%/上限50%

# 本月操作

{% asset_img 4.PNG %}

本月行情氛围高涨，月初一波暴涨，逢高减仓指数基金，逐渐向上个月开仓的主动基金富国天惠转移。

下半月暴涨过后，横盘震荡半个月，两次单日暴跌，都进行了小幅结余定投，毕竟大盘这个点位，单日暴跌幅度再大，也远比月初高，算不上是黄金坑，保持谨慎小幅买入甚至不买更好。白酒估值已经到了止盈位，清空剩余4层落袋止盈。

本月新债中了三只，加上上个月没上市的一只，上市卖掉了两只，一顿超级大肉和一顿海底捞美滋滋。还剩两只，应该下个月上。

本月港股打新中了一手华夏视听和一手渤海银行，前者挣了一千多，后者赔个手续费，总体终于实现盈利了，起码钱没白转出去。还是得随缘打大肉，中签当中奖，妖股还是不碰为妙。

# 开支总结

本月收入正常，工资奖金无变化，无额外收入。

本月没有大额支出，其它日常开销正常。

结余出工资的80%投入资产配置。

# 次月策略

下个月如果没有大问题，多半还是躺着，最近事情特别多，连月末总结都拖了一周，其实也没啥好总结的，工作上面忙了，投资这方面就是被动躺着，倒也避免了很多短期波动。其实如果长期看好手持的行业企业，短期波动又如何，长期持有，少给券商打工，多投入时间给工作才是正道。

具体来说，下个月应该是和这个月一样，不出现大波动的话暂时不动，半仓运作上下都有空间调整仓位。

# 一些感想

没什么感想，因为这个月太忙，根本没做投资相关的研究学习，外加月初暴涨，再次陷入涨了就学不进去知识的心态里。本月总体在进行被动指数基金转主动基金的操作，感觉效果还不错。下个月有空的话再写点感想吧。
